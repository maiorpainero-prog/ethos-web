# ETHOS | Эволюция не любит послушных

Сайт-лендинг по этологии и психологии отношений.

## Установка

1. Скачайте репозиторий.
2. Установите зависимости:
```bash
npm install
```
3. Запустите проект:
```bash
npm run dev
```

## Технологии
- React
- Tailwind CSS v4
- Framer Motion (motion)
- Lucide React
- Vite
